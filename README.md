# Blog-Project

## 목적

- Nextjs 13버전을 활용한 개인 블로그 만들기

## 배포

https://soyeah-log.vercel.app/

## 구현기능

- md viewer를 활용한 생성된 게시글 보기
- md editor를 활용한 게시글 생성
- firebase를 활용한 DB
